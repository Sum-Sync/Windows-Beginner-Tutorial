# Windows-Beginner-Tutorial
它会是一个引导PC新手快速上手Windows操作的程序,我们希望每个人都可以享受到现代Windows的便利操作而不用担心操作流程上的问题
It will be a program that guides PC beginners to quickly get started with Windows operations. We hope everyone can enjoy the convenient operations of modern Windows without worrying about the issues in the operating process for beginners.
